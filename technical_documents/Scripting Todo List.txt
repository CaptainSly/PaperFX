Lua Scripting Documentation, written as it's created. 

GLOBALS

Underneath the _G Lua Global

paper - A Table holding most of the Global Paper Objects/Functions
class - A utility java class that allows you to create classes using the syntax outlined in the LuaSimpleClass.txt file

Underneath the Paper table

_VERSION - The Current Version of the Paper Engine | String
database - The Current Loaded Plugin Database | Java Object
ppl - The Plugin Loader, gives access to the current plugin list | Java Object
ini - The Ini File, gives access to the game to create custom settings | Java Object
location - The Location Property, gives access to switch the location
